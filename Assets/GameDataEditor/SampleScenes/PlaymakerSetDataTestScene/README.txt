Before running the GDE Playmaker Set Data Test Scene do the following:

1) Make sure GDE Playmaker is enabled: (http://gamedataeditor.com/docs/enable-playmaker.html)
Go to File→Build Settings→Player Settings for the platform you are using and add “GDE_PLAYMAKER_SUPPORT” to the Scripting Define Symbols.
  
2) Import the Playmaker Globals for this scene.
Click SetData_PlaymakerGlobals_EXPORTED.asset in your Project Explorer. At the bottom of the Inspector window, click Import Globals.
